FAQ

Q: How do I add a task?

A: Clicking the + icon next to the table will create a new task you can directly type into

Q: How do I update a task?

A: Clicking the ^ icon next to the table will open a textbox to easily edit the task

Q: I want to make a custom table tmplate, but I don't know how to

A: The console allows you to create custom templates by typing "new template" template_name col_1 col_2 col_3
